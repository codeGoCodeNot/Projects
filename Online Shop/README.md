# Online Shop Documentation

## What is this?

A frontend-only demo shop.  
You can browse products and click on stuff.  
There is no real shopping, no backend, no data saved, and no actual checkout.

## Live Demo

See it running here:  
[https://tangerine-starburst-bc4628.netlify.app](https://tangerine-starburst-bc4628.netlify.app)

## Main Files

- `src/OnlineShop/ProductsGrid.tsx` — shows the grid of products.
- `src/OnlineShop/Product.tsx` — single product card.
- `src/OnlineShop/types.ts` — product types.

## How To Run

```bash
git clone https://github.com/codeGoCodeNot/Projects.git
cd Projects/Online\ Shop
npm install
npm start
```

Open your browser at [http://localhost:3000](http://localhost:3000).

## Features

- Search products.
- Click products to view details.
- Nothing gets saved.
- No login, no payments, no backend.

## Stack

- React
- TypeScript
- React Router

## For Devs

- All data is local, in memory.
- Change `products` array in code if you want new stuff.
- Use this as template or playground. No real use beyond demo/testing.

---

That’s it. No magic. Just run and click around.
